# Real-Time Chat Application

A full-stack real-time chat application built with **React**, **Node.js**, **Express**, **MySQL**, and **Socket.IO**.  
Users can create channels, join existing ones, and exchange messages in real time.

- Real-time messaging powered by **Socket.IO**
- User authentication and session management (JWT)
- Create and join **public** or **private** channels
- View list of **channel members** with avatars
- Persistent chat history stored in MySQL
- Modern, responsive UI built with pure CSS and React
